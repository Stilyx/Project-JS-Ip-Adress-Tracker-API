<h2>Inspiração do Desafio</h2>

<h2>Sobre</h2>

<p> Esse Site serve para gerar a localização no mapa  de um determinado IP.

<h2>Técnologias Utilizadas</h2>

- HTML
- CSS
- Javascript
- API

<h2>APIS</h2>

- Leaflet
- IP Geolocation API

<p>O Leaflet serve para Mostrarmos um mapa através de uma determinada longitude e latitude</p>

<p>o IP Geolocation API serve para pegar informações de um determinado IP. Usei apenas as:</p>

- IP
- Latitude
- Longitude
- TimeZone
- ISP
